# System Architecture Overview

## Overview

This diagram represents the simplified technology environment used by GoldLeaf Accounting LLC.

Employees work remotely from home offices and connect to company systems through the internet.

---

## Architecture Diagram

```
Employee Laptop
      │
      ▼
Home WiFi Network
      │
      ▼
   Internet
      │
      ▼
+-----------------------+
|   Cloud Services      |
|                       |
|  - Email System       |
|  - Cloud File Storage |
|  - Accounting System  |
+-----------------------+
```

---

## Description

Employees use laptops connected to their home WiFi networks.
These systems access company resources through cloud services hosted on the internet.

Because the company relies heavily on cloud platforms and remote access, protecting user credentials and endpoints is critical.
