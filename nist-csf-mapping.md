# NIST Cybersecurity Framework Mapping

## Overview

This document maps the identified cybersecurity risks at GoldLeaf Accounting LLC to relevant categories in the NIST Cybersecurity Framework (CSF).

The goal is to demonstrate how the organization's security improvements align with industry cybersecurity best practices.

---

## NIST CSF Functions

The NIST Cybersecurity Framework consists of five core functions:

* Identify
* Protect
* Detect
* Respond
* Recover

---

## Control Mapping

### Risk: Phishing and Credential Theft

Relevant NIST CSF Functions:

Protect

Recommended Controls:

* Multi-Factor Authentication (MFA)
* Security awareness training
* Email filtering protections

---

### Risk: Ransomware Infection

Relevant NIST CSF Functions:

Protect
Detect
Recover

Recommended Controls:

* Endpoint protection software
* Regular data backups
* System patching

---

### Risk: Weak Passwords

Relevant NIST CSF Functions:

Protect

Recommended Controls:

* Strong password policies
* Password manager usage
* MFA implementation

---

### Risk: Insecure Home WiFi

Relevant NIST CSF Functions:

Protect

Recommended Controls:

* WPA2 or WPA3 encryption
* Router security guidance
* Firmware updates

---

### Risk: Malware Infection

Relevant NIST CSF Functions:

Protect
Detect

Recommended Controls:

* Endpoint protection
* Software installation restrictions
* Security monitoring
